# 04 — Todo List

## Problem

Build a todo list where the user can:
- Add a new task by typing and pressing Enter or clicking Add
- Mark a task as done (strikethrough)
- Delete a task
- See a count of remaining tasks

## What to focus on
- Managing state as an array of objects (not just strings)
- `form` submit with `e.preventDefault()`
- Toggling a property on an item without mutating the original array
- Filtering out deleted items
- Keeping the UI in sync with state

## Run it

Open `index.html` in a browser.
