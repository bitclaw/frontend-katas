# 05 — Fetch & Filter

## Problem

Fetch a list of users from a public API, display them, and let the user filter by name in real time. Show a loading
state while the data is being fetched and handle errors gracefully.

Uses: `https://jsonplaceholder.typicode.com/users`

## What to focus on
- `async/await` with `fetch`
- Loading and error states
- Filtering on fetched data (not a local array)
- Keeping the full list in memory so you can reset the filter

## Run it

Open `index.html` in a browser. Needs an internet connection to hit the public API.
